## Design Features

- Heritage Gold double-border layout (cream parchment, maroon & gold accents)
- Hindi + English bilingual labels (standard Indian biodata format)
- Sections: Personal Details, Education & Career, Family Background, Address, Partner Preferences, Photos
- Photo placeholders (main + 4 gallery slots) — no images included yet
- Print / Save as PDF button (use browser Print → Save as PDF)
- Mobile responsive · WhatsApp-share friendly

## Before Sharing — Update These

Search for **"To be updated"** in both HTML files:

- Date of Birth, Time of Birth, Place of Birth
- Height, Complexion, Blood Group
- Gotra, Rashi, Nakshatra, Manglik status
- Phone number in Contact section

### Add Photos

1. Add image files to this folder (e.g. `aman-photo.jpg`, `nikita-photo.jpg`)
2. In each HTML file, uncomment the `<img>` tag inside `.photo-main`
3. Replace gallery `.photo-thumb` divs with `<img>` tags as needed

Repo: [github.com/amanazm/bio-data](https://github.com/amanazm/bio-data)

2. **Settings → Pages → Branch: `main` → / (root)**
